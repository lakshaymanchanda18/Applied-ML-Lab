# EXP1

## Experiment 1: Introduction to Applied Machine Learning

### Objective
The objective of this experiment is to understand the basic workflow of an Applied Machine Learning task, including data loading, preprocessing, model training, and evaluation.

### Problem Statement
Develop a simple machine learning pipeline to analyze a dataset and build a predictive model. The experiment should demonstrate:
- Data acquisition
- Data cleaning and preprocessing
- Feature selection or transformation
- Model training
- Model evaluation

### Requirements
- Python 3.x
- Jupyter Notebook
- Required libraries such as:
  - NumPy
  - Pandas
  - Matplotlib
  - Seaborn
  - Scikit-learn

### Procedure
1. Import the necessary libraries.
2. Load the dataset.
3. Inspect the data and handle missing values.
4. Perform preprocessing and feature engineering.
5. Split the dataset into training and testing sets.
6. Train a suitable machine learning model.
7. Evaluate the model using appropriate metrics.

### Expected Outcome
A working machine learning workflow that produces meaningful predictions and evaluation results.

### Conclusion
This experiment provides a foundational understanding of how machine learning models are built and assessed in practice.
